## PNN10

### Mobile phone number

The number can be this format

- xxxxxxxx
- +225xxxxxxxx
- (+225)xxxxxxxx
- (00225)xxxxxxxx
- (00225)xxxxxxxx

<code>
const pnn10 =  require("./index");
</code>
<br>

<code>
    var number = "(+225) 44 17 56 31";
</code>
<br>
<br>

<code>
    var converted_number = pnn10.phone_number(number);
</code>
<br>
<br>


### Fixe number
<code>
    var user_fixe_number = "20 30 09 09";
</code>
<br>

<code>
var converted_fixe_number =  pnn10.fixe_number(user_fixe_number);
console.log(converted_fixe_number);
